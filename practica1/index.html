Circuito con temporizador 555

Andrea Tapia Balvanera
Ingeniería
Fecha: Agosto 16, 2024
Arduino Electrónica

- Resumen -

En la primera práctica, se diseñó y construyó un circuito astable utilizando el temporizador 555. El circuito tiene como objetivo hacer parpadear un led, para ello, se hizo uso de un potenciométro y una fuente de alimentación de 9V.

- Introducción -

El temporizador 555 es uno de los circuitos integrados más populares y versátiles en el mundo de la electrónica. este componente se ha utilizado en una variedad de aplicaciones debido a su capacidad para funcionar en diferentes modos: astable, monoestable y bistable. Cada modo permite al 555 realizar diferentes funciones, desde generar pulsos hasta crear temporizaciones precisas.

En esta práctica, se empleó el temporizador 555 en configuración astable para crear un circuito que produce un parpadeo de LED. La frecuencia del parpadeo se regula mediante un potenciómetro, permitiendo ajustar la velocidad del parpadeo del LED.

- Materiales -

Los materiales que se utilizaron fueron: leds, un temporizador 555, un capacitor, un transistor, jumpers macho macho, un par de caimanes, una pila de 9V, resistencias de 220 ohms y 1KOhms, potenciómetro.

- Desarrollo -

Hicimos la práctica utilizando como guía el diagrama del circuito astable. Utilizamos la plataforma de tinkercad para simular el circuito basándonos en el diagrama mencionado, e identificar cómo funcionaba para asegurar que lo construyéramos correctamente.

Una vez realizado el circuito en tinkercad, construimos el circuito en físico. Para esto seleccionamos las piezas que íbamos a ocupar (jumpers, resistencias, leds, capacitores, potenciómetro) y empezamos con el armado. Por último, conectamos la tierra y la línea de VDD a las terminales de la pila para alimentar el circuito.

Simulación

Antes de pasar a la construcción física del circuito, se simuló en tinkercad. Para esto, se siguió el diagrama del circuito astable. La simulación permitió verificar el funcionamiento del circuito y confirmar que el temporizador 555 operaba como se esperaba, generando el parpadeo del LED con la frecuencia ajustable.

Gracias a la simulación, fuimos capaces de entender el comportamiento del circuito y así, hacer los cambios necesarios en caso de que no funcionara. Esta fase de simulación facilitó una comprensión más profunda del circuito y aseguró que todos los componentes estuvieran correctamente configurados.

Diseño

Se siguió el diagrama del circuito proporcionado en clase. El circuito está configurado en modo astable, lo que significa que marcaba altos y bajos de forma continua. Para esto se hicieron uso de tres resistencias importantes (de1000 ohms y 220 ohms), un capacitor, transistor y un potenciómetro.

Construcción

Se hizo uso de un protoboard para la base de la construcción de nuestro circuito. Los componentes fueron conectados según el diagrama, haciendo uso del temporizador, capacitor, transistor, resietencias, led. Las conexiones se realizaron con cuidado para asegurar la correcta integración de cada componente. Finalmente, las líneas de alimentación y tierra fueron conectadas a una pila de 9V, proporcionando la energía necesaria para el funcionamiento del circuito. Se verificaron todas las conexiones antes de encender el circuito para asegurarse de que no hubiera errores y que todo estuviera funcionando según el diseño.

- Resultados -

El circuito funcionó de manera correcta. El led parpadeó y se podía ajustar con el potenciómetro, coincidiendo con la simulación.

- Conclusiones -

La práctica permitió comprender el funcionamiento del temporizador 555 en configuración astable y su aplicación en la generación de señales oscilatorias. La simulación previa a la construcción fue útil para prever el comportamiento del circuito y realizar ajustes necesarios. La construcción física confirmó que el diseño y la simulación estaban correctos.
