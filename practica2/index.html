Creación página html

Andrea Tapia Balvanera
Ingeniería
Fecha: Agosto 23, 2024
Programación HTML

- Resumen -

En esta práctica aprendimos a programar una página en el programa de Visual Studio Code, utilizando el lenguaje HTML, que nos ayuda a crear páginas web. Aprendimos a organizar el contenido, estructura y formato de la página web mediante el uso de diferentes tamaños de letras, imágenes y uso de hipervínculos.

- Introducción -

HTML proporciona una forma de organizar y representar información en una página web, utilizando “marcas” o “etiquetas” para señalar la función de los elementos como texto, imágenes multimedia, etc. Durante la sesión, creamos y estructuramos un documento HTML sencillo usando etiquetas como. Las etiquetas se utilizan para organizar la información de manera lógica, lo que facilita su representación en un navegador y su vinculación a través de enlaces. En esta práctica, utilizamos el programa de Visual Studio Code y aprendimos la función de las etiquetas para organizar nuestra página.

- Materiales -

Los materiales que se utilizaron fueron: una laptop con la aplicación de visual studio code y acceso a la web.

- Desarrollo -

Para el desarrollo de la práctica, se trabajo en la aplicación visual studio code. Primero, comenzamos con la estructura básica del documento HTML. El código inicia con la declaración "!DOCTYPE html", que indica que estamos utilizando la versión HTML5. Después de esta declaración, incluimos las etiquetas "html", que encierran todo el contenido de la página. Dentro de la etiqueta, se encuentran las secciones "head" y "body", donde colocamos el contenido visible de la página. Comenzamos con varios títulos, utilizando las etiquetas "h1", "h2", "h3", y "h4", que definen un tamaño de encabezado diferente, siendo "h1" el más grande en el que usamos el título "INDEX".

Después de los encabezados, incluimos un párrafo de texto utilizando la etiqueta "p". En este párrafo, aplicamos algunos estilos en línea directamente dentro de la etiqueta, usando CSS. Definimos el color del texto en rojo, establecimos el tamaño de la fuente en 10 píxeles, y seleccionamos la fuente "Arial", "Helvetica" y "sans-serif". Esto nos permitió personalizar la apariencia del texto. A continuación, agregamos una imagen a la página utilizando la etiqueta "img". Indicamos la ruta de la imagen como "imagenes/gimnasio.jpeg" y cambiamos su tamaño a 100 píxeles de ancho.

Finalmente, incluimos un enlace a otra página utilizando la etiqueta "a", que hace un vínculo a otra página HTML. El enlace lleva a un archivo llamado "pagina2.html" y muestra el texto "PAGINA 2".

Simulación

- Resultados -

Nuestra página quedo organizada conforme el tamaño de los encabezados y utilizamos diferentes colores y formatos de letra para darle formato a nuestro contenido. También modificamos el tamaño de nuestra imagen y creamos un hipervínculo que nos lleva directamente a otra página.
